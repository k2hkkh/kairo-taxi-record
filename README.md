# 🚖 택시 운행 기록장

**카이로 AI 연구소 디자인 원칙을 따른다**

---

## 📱 소개

택시 운행 기록장은 **설명 없이 바로 쓸 수 있는** 택시 기사를 위한 웹 앱입니다.

- ✅ 회원가입 불필요
- ✅ 인터넷 연결 불필요 (최초 1회만)
- ✅ 휴대폰 브라우저에서 바로 사용
- ✅ 개인택시와 법인택시 모두 지원
- ✅ 기록 자동 저장
- ✅ 월별 달력으로 한눈에 확인
- ✅ 엑셀로 내보내기

---

## 🚀 시작하기

### 1단계: 웹사이트 열기
다음 링크를 휴대폰 브라우저에서 열어주세요:
```
https://k2hkkh.github.io/kairo-taxi-record/
```

### 2단계: 홈 화면에 추가 (선택사항)
더 편하게 쓰려면 홈 화면에 추가하세요.

**iPhone:**
1. Safari에서 열기
2. 아래 공유 버튼 → "홈 화면에 추가"

**Android:**
1. Chrome에서 열기
2. 메뉴 → "홈 화면에 추가"

### 3단계: 사용 시작
- 택시 유형 선택 (개인/법인)
- 오늘의 매출과 비용 입력
- 저장 버튼 클릭 → 완료!

---

## ✨ 주요 기능

### 📊 오늘 기록 입력
- 💰 **매출**: 오늘 벌어들인 금액
- ⛽ **주유비**: 휘발유 또는 LPG 비용
- 💳 **기타비용**: 식비, 수리비 등 (항목 + 금액)
- 📝 **메모**: 특이사항 기록

### 📈 자동 계산
- **오늘 순수익** = 매출 - 주유비 - 기타비용 (실시간 계산)
- **월별 통계**: 총 매출, 총 비용, 순수익 자동 계산

### 📅 월별 달력
- 기록이 있는 날짜에 순수익 표시
- 날짜 클릭해서 상세 정보 확인
- 이전/다음 달 이동

### 🏢 법인택시 기능
- 차량번호, 사번 최초 1회 입력 (이후 자동)
- 운행 건수 기록
- 월별 총 운행 건수 자동 계산

### 💾 데이터 관리
- **자동 저장**: 기록이 휴대폰에 자동 저장
- **엑셀로 보관**: CSV 파일로 내보내기
- **백업**: 기록 백업 및 복구

1. **유치원생도 사용할 만큼 쉽다**

7. **사용자가 생각하지 않게 만든다**

## 💡 사용 팁

### 💾 정기적으로 백업하세요
- 메뉴 → "백업 불러오기"에서 백업 파일 다운로드
- 휴대폰을 바꿀 때 새 휴대폰에서 복구 가능

### 🗑️ 휴무일 설정
- 달력에서 날짜 클릭
- "휴무일 설정" 선택
- 빨간색으로 표시됨

### 📊 엑셀에서 보기
- 메뉴 → "엑셀로 보관"
- 다운로드된 파일을 엑셀에서 열기
- 회사 제출용으로 인쇄 가능

---

## ⚠️ 중요 안내

### 데이터 보관 방식
- 모든 기록은 **당신의 휴대폰 안에만** 저장됩니다
- 서버에 저장되지 않습니다
- 인터넷 연결이 없어도 불러올 수 있습니다

### 주의사항
1. **브라우저 캐시 삭제 시 기록이 사라질 수 있습니다**
   - 정기적으로 백업하세요

2. **휴대폰을 바꿀 때**
   - 백업 파일을 새 휴대폰으로 옮겨서 복구하세요

3. **여러 기기에서 쓰려면**
   - 각 기기에서 별도로 백업/복구 필요
   - (클라우드 동기화는 준비 중)
